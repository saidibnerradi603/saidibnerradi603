## About Me

I'm an AI & Backend Developer with a Master's degree in Big Data and Decision Support. My work sits at the intersection of Generative AI and backend engineering — taking LLMs from experimental scripts and deploying them into functional, production-ready software architectures.

My current focus spans Retrieval-Augmented Generation, multi-agent systems, and LLM-powered automation. I engineer the REST APIs, vector database architectures, and event-driven backend infrastructure required to deploy LLMs securely, observably, and at scale in production environments.



## What I Build

| Area | Description |
|------|-------------|
| **Production-Ready RAG Systems** | End-to-end document retrieval and Q&A pipelines connected to vector databases |
| **Scalable AI Backends** | High-performance RESTful APIs (FastAPI) that seamlessly serve AI models to front-end applications |
| **Multi-Agent Workflows** | Event-driven, automated AI agents that interact with external tools and databases |
| **Automated Data Pipelines** | Robust web scraping and custom ETL architectures that transform raw data into actionable insights |
